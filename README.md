# 🐧 Miniguia de Linux para Iniciantes com IA

## 🎯 Contexto e Objetivos

Este projeto tem como objetivo estudar Linux para iniciantes utilizando Inteligência Artificial através do NotebookLM.

Objetivos:
- Aprender os conceitos básicos do Linux
- Entender o uso do terminal
- Aprender comandos essenciais
- Melhorar o uso de IA para estudo

## 📖 Miniguia de Estudo

### O que é Linux?
Linux é um sistema operacional open source, conhecido por sua segurança e flexibilidade.

---

### Comandos básicos:
- ls → listar arquivos  
- cd → navegar entre pastas  
- mkdir → criar pasta  
- rm → remover arquivos  

---

### Glossário:
- Terminal → onde você digita comandos  
- Shell → interpretador de comandos  
- Root → usuário administrador  

---

### Prompts reutilizáveis:
- Explique Linux para iniciantes  
- Liste comandos com exemplos  
- Crie um resumo sobre Linux  

---

## 🚀 Conclusão

Este projeto mostrou como usar IA para aprender Linux de forma prática e organizada.
