## 🧠 Engenharia de Prompts

### 🔹 Prompt 1:
"Explique Linux"

**Problema:**
Resposta muito genérica e superficial

**Melhoria aplicada:**
"Explique o que é Linux para iniciantes, usando exemplos do dia a dia e linguagem simples"

**Resultado:**
A resposta passou a ser mais didática, com exemplos práticos, facilitando o entendimento.

---

### 🔹 Prompt 2:
"Liste comandos Linux"

**Problema:**
A resposta trouxe apenas nomes de comandos sem explicação

**Melhoria aplicada:**
"Liste os 10 principais comandos do Linux para iniciantes, explique cada um e dê exemplos práticos de uso no terminal"

**Resultado:**
A IA retornou uma lista estruturada com explicações e exemplos reais, melhorando significativamente o aprendizado.

---

### 💡 Análise Técnica

- Prompts genéricos geram respostas superficiais
- Prompts detalhados aumentam a qualidade da resposta
- Incluir exemplos práticos melhora o aprendizado
- A engenharia de prompts é essencial para extrair valor da IA

## 📖 Miniguia de Estudo

### 📌 O que é Linux?
Linux é um sistema operacional open source amplamente utilizado por sua segurança, estabilidade e flexibilidade.

---

### 📌 Terminal
O terminal permite interação direta com o sistema por meio de comandos, sendo essencial para administração e uso avançado.

---

### 📌 Comandos Essenciais

| Comando | Função | Exemplo |
|--------|--------|--------|
| ls | Listar arquivos | ls -l |
| cd | Navegar entre diretórios | cd /home |
| pwd | Mostrar diretório atual | pwd |
| mkdir | Criar diretório | mkdir projetos |
| rm | Remover arquivos | rm arquivo.txt |

---

### 📌 Estrutura de Diretórios

- / → diretório raiz  
- /home → arquivos dos usuários  
- /etc → configurações  
- /bin → comandos essenciais  

---

### 📌 Glossário

- Terminal → interface de linha de comando  
- Shell → interpretador de comandos  
- Root → usuário com acesso total  
- Diretório → estrutura de arquivos  

---

### 📌 Prompts Reutilizáveis

- Explique Linux para iniciantes com exemplos práticos  
- Liste comandos Linux com exemplos  
- Explique a estrutura de diretórios  
- Crie um resumo técnico sobre Linux
